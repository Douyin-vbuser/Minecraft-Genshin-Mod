# 改进的落叶飘落轨迹方程组

设 t 为时间（秒），方程组如下：

## x 方向（左右运动）
x(t) = A?sin(ω?t) + A?sin(ω?t) + v_x * t + W_x * sin(ω_wt)

## y 方向（前后运动）
y(t) = B?cos(ω?t) + B?cos(ω?t) + v_y * t + W_y * cos(ω_wt)

## z 方向（垂直运动）
z(t) = H - (g * t? / 2) * (1 - e^(-k_zt)) + C?sin(ω?t)e^(-k_zt) + C?cos(ω?t)e^(-k_zt)

其中：
- A?, A?, B?, B?, C?, C? 是振幅参数
- ω?, ω?, ω?, ω?, ω?, ω? 是角频率参数
- v_x, v_y 是初始速度分量
- W_x, W_y 是风力影响的振幅
- ω_w 是风的变化频率
- H 是初始高度
- g 是重力加速度（约 9.8 m/s?）
- k_z 是空气阻力系数

## 叶子的旋转（可选，用于 3D 可视化）
θ(t) = θ? + ω_θ * t + D * sin(ω_θt)

其中：
- θ? 是初始角度
- ω_θ 是平均旋转角速度
- D 是旋转振幅
