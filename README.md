# MCI重构项目

由于原有项目的架构设计不合理，导致项目难以维护，因此决定对项目进行重构。

## 结构

 **·com.vbuser.genshin**:主模组,包含需要的方块、物品、剧情演出实现、大世界探索设计实现。
 
 **·com.vbuser.inventory**:仓库实现，用于实现仓库系统，提供增加、读取、删除物品，对不可堆叠物品追加备注（原版中的NBT标签）的功能。
 
 **·com.vbuser.browser**:浏览器实现，用于CG播放和后续版本活动小游戏、七神召唤接口提供。
 
 **·com.vbuser.movement**:实现自定义玩家实体运动规则、镜头运动规则。
 
## 原项目资产处理

原项目中材质会全部替换为官方版本，其余基本保留；代码基本全部舍弃，实现全部重写，保留主模组的理想境框架。

原项目中的用户协议、借物表、待办表全部舍弃，保留“不受欢迎的创作者”清单，启动器舍弃（计划魔改PCL2）
