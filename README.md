## 综述

项目的目的是以本模组为核心，配合Minecraft地图、指令与其他模组实现在Minecraft中尽可能完整地体验原神。

受限于作者的学业影响，单人开发会消耗很长时间，所以作者十分希望能得到国内外Minecraft与原神玩家和开源社区的帮助与支持。预计开发时长两年半

第一阶段计划完成现阶段（原神3.0版本）开放的**蒙德地区**，任务与剧情，主要难点在于得到蒙德地区的*地形测绘数据以及角色、配队、圣遗物系统的实装与元素、伤害系统的重写*。

## Overview
The purpose of the project is to take this mod as the core, cooperate with Minecraft world, commands and other mods to realize an as completed as possible experience of Genshin Impact in Minecraft.

Because of the fierce competition in China's college entrance examination (especially in Jiangsu Province), I have no time to complete this project quickly and with high quality. This project needs the help of the open source community and the player groups.

We plan to complete mondstadt first, including CG animations and plots. At present, the main challenge is to complete the role system,artifact system,elemental reaction system and the battle system.(because they are so different with Minecraft)
