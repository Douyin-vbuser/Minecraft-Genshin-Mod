# 写在离别之前：一个互联网透明人的自述

写下这段文字的时间，是2024/07/30。

写下这段文字的人，我，vbuser，一个原神、Minecraft双修玩家，MCI项目目前唯一的contributor。

请原谅我的自私，在项目的介绍文档前插入这一段主观的、只属于我个人的文字。

Minecraft-Genshin-Mod，亦称Minecraft Impact(故有简称MCI)在GitHub上，向来只是一个名不见经传的小项目，重构版只有不到6000行有效代码，资源文件几乎没有开始制作。对于“fork原神”这个远大的目标而言，这不过杯水车薪。

中考备考的一年，我在国庆假期最后一次在抖音上更新视频（是一个挖区块的延时摄影，用于批判爱玩游戏的赫某人新视频夸大肝度）。此后的很长一段时间，我都在思考未来视频更新的方向：工作流已经基本从封闭的苹果系移动端转向了PC，理论上我有能力做出更多方向上的尝试，也可以完成很多更为自由的创意。但是，具体要做些什么？PVP是不能打的，我玩游戏的历史还没有学编程的历史久远；红石音乐是烂完了的（仅针对当时作者眼中的抖音），你没有办法完成观众的留存，一首歌的时间，对于短视频而言，还是太长了些；建筑和红石虽然会，但是深入学习的成本也是巨大的，没有办法在暑假两个月转化为视频产出。

那怎么办？当时我的回答是选择开发模组。虽然Minecraft模组生态已经几乎饱和，有关模组开发的高播放量视频任然是一片蓝海，尽管专业性的文档性视频没有热度，但是鬼畜性的视频数据十分优秀，我自己当时也很喜欢拿出来看。于此同时，产生了第二个问题：模组开发能力只是工具，你又要做赋予他什么样的内核？

我当时给出了两个回答：一方面，继续完善自研的音符盒模组，争取早日实现自动化；另一方面，5月14日，距离中考一个月时，MCI正式立项。

为什么选择做原神？当时我的想法也算简单。一方面，当时已有的原神模组只是含有原神要素（可参考籽岷的视频），不是我所期望的作品，另一方面，当时根据抖音上零星刷到的原神相关视频，我也可以与之“对上电波”，它确实吸引我，我也确实想乘机获取流量。做视频吗，为流量，不寒碜。

当我怀着希望考完中考安心在家摆烂，用自研的btnote做好了一些列原神的红石音乐准备依此发布至抖音时，美梦破碎了。几乎一年没有更新，加之为了防止掉粉无底线互关，我的粉丝画像已经歪到不成样子，视频数据也必然是一团糟，主题曲的视频下面唯一一条评论更是刺痛人心

```
小op
```

这何尝不体现语言的魅力啊，简短，又如此有力。

我不知道如果同样的境遇发生在了其他彼时的新兴原神相关内容创作者身上（当然也一定发生在了他们之中一些人的身上）他们会如何应对，我做出的选择，是逃避。

删除了几个数据极差的视频之后，我注销了那个账户，那个记录着我初中三年积累Minecraft领域视频制作知识的载体。事实上，也确实是彼时我对Minecraft圈子的失望的集中体现：自从Dream的视频被搬运到国内爆火后，一段时间内抖音上几乎看不到其他种类的MC视频（那时还没开始用B站，具体什么情况也不知道）。彼时我总结了当时我认为的抖音Minecraft社区的几大罪状：
```
1.乱搞个人崇拜，尤其是对实况主（我当然理解这些视频主本人没有错，这都不是他们所能控制的）
2.外网内容搬运横行，原创吃力不讨好，数据远比不上营销号
3.我期望我做出有些质量的内容，但是抖音推流机制决定了你必须高频率更新
4.抖音强推DOU+服务，明目张胆搞流量绑定资本，与我做视频的本心相悖
5.抖音长期受B站用户歧视，你被骂的原因可能只是你国际版Minecraft的ID里顶了个Douyin，尽管你没有做错任何事
```

当时与我而言，没有选择，6月22日下完原神，6月23日开始和YouLing_spector玩原神后，我几乎没有再想过任何和MC相关的事情。

这种情况一直持续到了8月底，军训结束，马上就要迎接高中生活。这是改变我精神状态的一件事情，是原神与必胜客的联动，在我家附近正好就是一家联动的门店，说实话，那是我第一次看到那么多与我有着共同喜好，年龄相仿的人。尽管当天很热，但他们的精神面貌称得上充满朝气，这对于在家瘫了两个月的我而言，影响自然是巨大的。这时，也正是MCI这个项目启程的时间。

对于项目开发的过程而言，在这里我无意做出过多叙述，只不过是一个一边学习一边前进的过程，中途有过因学业的繁重而暂停，有过不得不放弃一条技术路线的心碎，有过因代码出现了无法解释的诡异现象的迷惘，有过万念俱灰，也有过重启的希望。

项目开始时，我清楚，这是扔出了一枚骰子，它将在空中至少飞行三年，三年的时间里，我可以拨动这枚骰子，但它走向何处，没有人会知道。

一场为期至少三年的豪赌，在此期间，社交媒体不可以能有任何产出（因为需要为后期积累足够视频资产，观众没有义务对看不到未来的共创项目伸出援手），到如今，大半时间过去，眼前是不得不经历的一年完全静默，你问我，后悔了吗？

我不知道。

对不起，对于这个问题，我无法做出回答。

三年混迹于抖音的历程，从ICE服务器被炸毁开始，到不更新爆火结束。我尝试过很多，经历了许多失败，我可以承认我技不如人，无法产出人们喜爱的作品。我接受因为我自己的愚行，导致浅度还原天空岛的视频数据远不如一个技术力不及我的视频有着比我好看的多的数据。但是，我不能接受的是抖音是一个社区向良性发展的最大敌人，抄袭的视频任你举报，举报完照样推流（指抖音创作者神马海王星抄袭B站up主白勺的Lemon，我本人举报后二度刷到同一视频）。我曾经认为是我没有用好抖音，就像一些人所宣称的是用户自己没有用好360，奇怪的是，自从我开始集中拉黑抖音营销号后，我几乎再也没有看到Minecraft内容，如果是这样，这又是谁的错。

短视频平台没有给一个以创造为核心的游戏一个良性的社区环境，我凭什么，又为什么要指望 改变这种情况。

短视频平台靠不上，所以我没有允许利用了本项目产生的影像在百度app、好看视频、抖音、西瓜、快手上不受顾虑地传播。

如此做来，对吗？

这倒是让我想起来第一次散兵剧情节奏时的键政命题：你创作的内容不是老百姓喜欢的内容，那你的作品就是脱离人民。那么，我们先来看看什么是“更广大人民群众所喜爱的内容”。抖音月活多少？畅销榜什么排名？你B站又月活多少？畅销榜什么排名？人人都知道短视频危害大，新黄色新闻制造了“浮桥案”“无锡蛋糕店老板性侵女童案”，又如何呢？23年，央视还敢抨击“鸡你太美”是恶俗烂梗（虽然这件事情央视做的不完全对，但也不是大错，但借**此**攻击其公信力倒是荒谬的）。24年呢？直接向“科目三”滑跪了。你央视在新媒体时代，理应作为对抗媚俗文化的重要力量，你自己向媚俗投降滑坡为“民俗”，你脸往哪里挂？所以让我重新评价短视频究竟如何？

我的答案是，简体中文互联网（我不会翻墙，没有调查就没有发言权）已经不可避免地、几乎无法阻止地滑向了**后现代状态**，我作为一个岌岌无名的内容创作者，我可以不认可，但是我不配说这是错的。

后疫情时代的时代特性是滋生后现代性的绝佳土壤，大资本为对抗风险阻滞财富的流动，国家政权刚刚经历浩劫，能做的事情很有限，与后现代作为“资本主义社会的终级阶段”是一致的。

那后现代社会的特征是什么：大他者的缺失、超我命令的享乐义务。

那同步到游戏社区中，结果是什么？

作者认为，一是“暴力”手段的再分配，此处暴力手段不全等于网络暴力但包括网络暴力，强调force的表意，任何人都可以原地对任何人在任何地方发表的任何观点提出批评，并且这种批评几乎没有下限，形式上甚至可以为辱骂、恶意解构且不需要为之付出被批评的代价(点名被导演BK采访的某一位)。

由此产生第二个结果：批评获得了自由，但是批评的批评倒是开始变得不自由了，这一点区别于经典后现代理论。《人群：群体性社会学》（亦称乌合之众，但作者认为该译名不恰当）中认为“批判行为”是绝对的进步行为。但是现代互联网毕竟不完全是几百年前哲学家们所构想的社会结构，互联网时代是后真相时代，真相掌握在少数参与者手中，至于其他人，他们所“知道”的真相其实并不知道，他们或许最后被验证为真，或许被验证为假，但并没有什么区别，他们只是做出了自己的立场选择。选择负面立场，并不全等于批判。

第三个结果，群体性造神和毁神运动。作者认为，后现代社会是一个“离历史太远，离过去太近”的历史阶段，人们对故纸堆里的记忆没什么兴趣，但是又没有成为“自由而完全发展的人”。换言之，人们需要偶像，但是后现代社会不容许大他者的存在。

最后，是垄断性大资本全身而退，后现代是比烂的时代，人们不对大资本抱有任何期待，也并因为他们无力做出反抗，同时，大资本可以利用自身对MCN的培养执行对新兴势力的做多和做空。需要注意的是，此处大资本的指向通常需要铺开极广阔的业务范围，而数字时代中国的互联网软件性生态，正是由他们建成的。

简而言之，作者认为，后现代社会是对“创业”者不利的社会，做up主，你可能就因为吐露了后期的艰难就造成了范围性的取关；后现代社会是大众内容向游戏/类电影作品发展不利的时代，后（现代）解构成本极低，产物精简，传播性影响极大，完全适配于碎片化时代的传播要求，同时，对大他者的不信任造成的造神-毁神周期是你自身无法掌控的。

最后再来谈谈作者对近期一些事件自己的看法吧。

一、ML运动。作者认为，事实上ML运动已经事实上失败了，ML运动早期核心诉求是什么，是要求厂商满足、重视其需求，理应产生的结果是ML向游戏的繁荣，而绝对不是仅有某一个ML游戏的成功。作者曾在B站看到视频《（原神）他改变了二游，她又改回去了》，当时作者的评价是，可不是嘛，原神顶住了舆论压力，证明了把心思放在游戏质量上可以扭转命运，“ta”可不就又给改回去了吗。后来作者又创建仓库MLMovementSus调查B站有关游戏的舆论动向，其结论更是令人咋舌：一个创作者激励计划携带率95+%的社区尽然好意思骂一个激励计划携带率70%~50%的社区搞孝子文化；一个注定见不得光的内容向尽然敢发出和大众化内容一样响亮的声音（指某二游的小时视频发布数在5月底到其激励计划结束与原神不相上下）。~~关于这个游戏，作者提出一个更为激进的猜测仅供娱乐。第一次看到这个游戏时作者想到了韩国的Nikkei，后得知该游戏试图进入国内但是未成功，但依然在海外产生巨大收益甚至与腾讯并表财报，因此，可以产生一种颇具阴谋论的猜测。~~

ML时期作者开发过爬虫，很可惜直到最后也没有对其自动化后挂载在服务器上，并且可以统计学性确定B站对项目中使用的游客Cookie的搜索结果进行污染。如果这个项目没有被删除，必定会对如今的”真相“认定产生巨大影响

这件事情还给作者带来了另一份启发，可能如今的游戏运营思路不是”让玩家的游戏被运营“而是”让玩家相信玩家的游戏被运营吧“（梗出自《是，首相》）

二、龙王事件。不可否认的，这是原神项目组犯下的巨大错误，那天作者看着自己的项目，盯着窗外发呆了很久，我与刚入坑绝区零的Question__说，”最好的结果，也许他们明天中午左右就会道歉回档，补偿十连（或者开放退款通道）“对于超长期二创项目而言，这个事件的毁灭性是巨大的，原神项目组应当清楚自己的处境：元末明初刚过去没多久，一股势力希望借鸣潮之手一举干掉米游的目的昭然若揭，一次没有成功，他们就会发起第二次、第三次；新游异环发布，互联网上对米游发表负面评论的事实性益处已经增大到不需要水军的阈值，为了TNO（指没有米游的新二游秩序），他们必然不惜一切能用的手段。而这个时候，你犯下事实性错误，就是”承认“所有的诋毁，无论如何，这都是个愚蠢至极的决定。

三、某三字up主事件。首先，我仅代表我作为观众个人表达我对其视频的看法：如果说某个up主上来就是以氪佬自居，那我大抵对他的视频不感兴趣。以影视飓风的内容需求倒金字塔而言（”创作者满意<甲方满意<观众满意“），你本可以做观众，却为什么自降身份为甲方？除非你自认为你需要一个满足”甲方满意>观众满意“的乙方来服务你。至于其他事件，我个人对其不感兴趣、未作了解。如果他遭遇了什么不幸，我深表同情，但仅此而已，又不是我干的。

至于为什么我厌恶氪佬映像，就简单引述一段话吧

>就消费来看，当消费脱离了对商品使用价值的诉求而沦为对符号的消费时，消费的后现代特征也就显露无疑了。后现代马克思主义者认为，通过符号消费，人们在消费中自然建立起一种根据消费符号序列而产生的相应身份认同。也就是说，当人们进行消费时，人与人之间的社会关系与社会地位就已经显露并展现出来......人们通过符号消费，实际上是在寻找相应群体归属与身份认同，这与之前人们通过劳动工作来建立身份认同形成了鲜明的对照。<br>——《消费异化批判 后现代马克思主义立足于人与社会的关系的异化》（P90）

说到最后，还是得感谢您能听完一个渺小的个体在奔赴另一端旅程前的絮叨，开发的这两年真的发生了很多，这份经历独一无二，我希望把它装进漂流瓶里，扔进名为互联网的大海里，也许有一天它会被拾起，也许它会就此消失，但无论如何，我们曾同做着一个梦，一个旅者、一份奇遇。

# MCI重构项目

由于原有项目的架构设计不合理，导致项目难以维护，因此决定对项目进行重构。

## 结构

 **·com.vbuser.genshin**:主模组,包含需要的方块、物品、剧情演出实现、大世界探索设计实现。
 
 **·com.vbuser.database**:简易数据库实现，用于实现背包系统，提供增加、读取、删除物品，对不可堆叠物品追加备注（原版中的NBT标签）等功能。
 
 **·com.vbuser.browser**:浏览器实现和http服务器搭建，用于CG播放和后续版本活动小游戏、七圣召唤接口提供。
 
 **·com.vbuser.movement**:实现自定义玩家实体运动规则、镜头运动规则。

 **·com.vbuser.particulate**:通用粒子、特效渲染控制器。部分需要自定义的渲染逻辑
 
## 原项目资产处理

原项目中item材质会全部替换为官方版本，其余基本保留；代码基本全部舍弃，实现全部重写，保留主模组的理想境框架。

原项目中的用户协议、借物表、待办表全部舍弃，保留“不受欢迎的创作者”清单，启动器舍弃重写。
